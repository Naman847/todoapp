# To-Do App

A simple yet complete **To-Do web application** built using **Spring Boot**, **Spring Data JPA**, and **Thymeleaf**.  
This project demonstrates core **MVC principles**, **database persistence**, and **CRUD operations** for managing daily tasks.

It serves as a foundational backend project that highlights essential skills in **Java web development**, **ORM with Hibernate**, and **server-side rendering**, with clear potential for further extensions such as authentication and REST API exposure.

---

## Features

- **Task Listing:** View all tasks with their completion status (`GET /tasks/all`).
- **Create Task:** Add new tasks via form submission (`POST /tasks/create`).
- **Toggle Completion:** Mark tasks as complete or incomplete (`POST /tasks/{id}/toggle`).
- **Delete Task:** Remove tasks permanently (`POST /tasks/{id}/delete`).
- **Persistent Storage:** All data stored in MySQL with automatic schema generation.
- **Server-Side UI:** Dynamic rendering using Thymeleaf templates.

## Database Schema

The database schema is auto-generated by Hibernate (`spring.jpa.hibernate.ddl-auto=update`).  
Below is the inferred structure:

```sql
CREATE TABLE task (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    description VARCHAR(255) NOT NULL,
    completed BOOLEAN NOT NULL DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
````

## Getting Started

### Prerequisites

* Java 24 or later
* MySQL / MariaDB (default DB: `todoapp`)
* Maven (or use the included wrapper)

### Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/yourusername/todoapp.git
   cd todoapp
   ```

2. **Configure Database**
   Update `src/main/resources/application.properties`:

   ```properties
   spring.datasource.url=jdbc:mysql://localhost:3306/todoapp
   spring.datasource.username=root
   spring.datasource.password=yourpassword
   spring.jpa.hibernate.ddl-auto=update
   spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQLDialect
   ```

3. **Run the Application**

   ```bash
   ./mvnw spring-boot:run
   ```

4. **Access in Browser**

   * Open: [http://localhost:8080/tasks/all](http://localhost:8080/tasks/all)
   * Use the interface to create, toggle, and delete tasks.
